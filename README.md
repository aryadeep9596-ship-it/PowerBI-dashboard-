# PowerBI-dashboard-
This is my first PowerBI dashboard project 

Project Overview
This project presents an interactive Sales Analysis Dashboard built using business intelligence tools to analyze global supermarket performance.
The dashboard provides insights into:
Total Sales Performance
Profitability Analysis
Regional Performance Comparison
Category & Sub-Category Contribution
Time-based Sales Trends
Segment-wise Sales Distribution

It helps stakeholders understand where revenue is coming from, which regions are performing best, and which product categories drive profit.
